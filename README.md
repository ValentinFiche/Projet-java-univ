# Projet-java-univ

Ce dépôt contient une série de **projets et exercices Java réalisés dans le cadre universitaire**, permettant de mettre en pratique les concepts fondamentaux de la **programmation orientée objet**, de la **gestion de projets Java** et des **tests unitaires**.

## Contenu

**Jeux simples :**
- **Morpion** (Tic-Tac-Toe)
- **Bataille Navale**

**Applications éducatives / exercices :**
- **Gestion bancaire**
- **Projets de tests unitaires avec Mockito**

**Mini-projets divers** pour renforcer les compétences en **Java** et **algorithmique**.

## Objectifs

- **Appliquer** les concepts de programmation orientée objet et de modularité.  
- **Travailler** sur des projets courts et ludiques pour consolider l’apprentissage de Java.  
- **Expérimenter** avec les tests unitaires et les bonnes pratiques de développement.
